# Architecture Decision Records (ADR)

Набор архитектурных решений проекта. Все ADR фиксируют принятые решения и их последствия.

## List

- [ADR-0001: Router Model and Handler Resolution](0001-router-model.md)
- [ADR-0002: Multi-Module Project Structure](0002-multi-module-structure.md)

## Notes

- Статус `Accepted` означает, что решение принято и является текущим архитектурным baseline.
- Ломающие изменения ADR должны сопровождаться новым ADR или явной правкой существующего с объяснением причин.
